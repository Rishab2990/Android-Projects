## 🌟 Technologies and Topics Covered

- **Kotlin**: The primary programming language for **Android development**.
  
- **XML**: Used for designing layouts.

- **Firebase**: For backend services, **authentication**, and **real-time databases**.

- **Notifications**: Implementation of local and push notifications.

- **Fragments**: Modular components of UI for better app navigation.

- **MVVM**: **Architecture pattern** for separating concerns.

- **Coroutines**: For managing background tasks and **asynchronous programming**.

- **Unit Testing**: Ensuring code **reliability** and **performance**.

- **Jetpack Compose**: Modern toolkit for building **native UI**.

- **Room Database**: For **local data storage** and management.

- **Retrofit**: For **networking** and **API calls**.

- **Dagger/Hilt**: **Dependency injection** frameworks for efficient code management.

- **Material Design**: Applying design principles for user-friendly interfaces.

- **AndroidX**: Jetpack libraries for modern **Android development**.

- **Constraint Layout**: For **responsive UI design**.
